# Home Psychologist · Мастер Кит

**ИИ-психолог на дому** — интерактивное веб-приложение, которое анализирует «жалобное письмо» пользователя
и определяет по методике «Мастер Кит»:

- **Уровень развития** (1–7) с визуальной шкалой
- **Застрявшие эмоции** (1–2 ведущих, с цитатой-доказательством из текста)
- **Эмоциональную яму** (жертва / спасатель / преследователь / зависимость / одиночество / неуверенность / безысходность)
- **2–4 точечные проработки** с пошаговыми инструкциями
- **Следующий маленький шаг** на сегодня

Приложение адаптировано под мобильные платформы (VK Mini App, RuStore):
mobile-first вёрстка, safe-area, корректный viewport meta.

## Технологии

- **Next.js 16** (App Router) + **TypeScript 5**
- **Tailwind CSS 4** + **shadcn/ui**
- **z-ai-web-dev-sdk** — LLM-анализ с строгим JSON-выводом и валидацией
- **Framer Motion** — анимации появления блоков
- Тёплая «терапевтическая» палитра (терракота + кремовый фон)
- Шрифты Inter + Fraunces

## Конфигурация LLM (Z.ai / GLM-4)

Для живого ИИ-анализа нужен API-ключ Z.ai. Есть два способа настройки:

### Локальная разработка — файл `.z-ai-config`

Создайте файл `.z-ai-config` в корне проекта (уже в `.gitignore`):

```json
{
  "baseUrl": "https://api.z.ai/api/paas/v4",
  "apiKey": "ВАШ_КЛЮЧ"
}
```

### Продакшн на Vercel — переменные окружения

В настройках Vercel-проекта (Settings → Environment Variables) добавьте:

- **`ZAI_API_KEY`** — ваш ключ Z.ai (обязательно)
- **`ZAI_BASE_URL`** — `https://api.z.ai/api/paas/v4` (опционально, по умолчанию уже этот)

После добавления переменных сделайте Redeploy в Vercel.

### Как получить бесплатный ключ Z.ai

1. Зарегистрируйтесь на **https://z.ai** (бесплатный тариф даёт ~$18 кредита)
2. Перейдите в **API Keys** → https://z.ai/manage/apikey
3. Создайте новый ключ, скопируйте его
4. Добавьте его как `ZAI_API_KEY` в Vercel Environment Variables

Без ключа на Vercel приложение вернёт ошибку 500 при попытке диагноза —
это ожидаемое поведение. Демо-версия на GitHub Pages работает без ключа.

## Деплой

Приложение готово к деплою на Vercel. Два варианта:

### Вариант 1 — One-click (самый простой)

1. Откройте **https://vercel.com/new**
2. Выберите репозиторий `evikass/home_psychologist`
3. Framework определится автоматически (Next.js)
4. Нажмите **Deploy** — через ~1 минуту получите ссылку вида `home-psychologist.vercel.app`

### Вариант 2 — Автодеплой через GitHub Actions

В репозитории уже настроен workflow `.github/workflows/deploy.yml`.
Чтобы он заработал, добавьте 3 секрета в Settings → Secrets and variables → Actions:

- `VERCEL_TOKEN` — создайте на https://vercel.com/account/tokens
- `VERCEL_ORG_ID` — найдёте в `.vercel/project.json` после первого ручного деплоя
- `VERCEL_PROJECT_ID` — там же

После этого каждый `git push` в `main` будет автоматически деплоить новую версию.

## Принципы методики

1. Эмоция — это энергия. Если её не проживать — она копится в теле и блокирует жизнь.
2. Любая «яма» держится на выгоде. Когда выгода осознана — выход открывается.
3. Ответственность = свобода. Где я автор — там я могу изменить.
4. Тело — компас. Любая проработка идёт через контакт с телом, не через голову.
